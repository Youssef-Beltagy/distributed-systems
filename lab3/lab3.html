1. Purpose

This laboratory work intends to let you understand how to code and run RMI programs and mobile agents, so that you can focus on your HW3's code design.

2. Statement of Work

Since we have less than 1 hour for Lab 3, let's focus on item 3: MyAgent.
  1. Read program 3's specification carefully to understand how to run RMI and UWAgent.
  2. Redesign the lab work 1's client/server programs with RMI. In other words, your client program should call the following server function: 
       double[] sqrt( double[] array )
    This function receives an array of doubles, updates each array element with its square root, and returns the modified array back to a client.
  3. Code a simple mobile agent, say MyAgent.java that migrates to uw1-320-20, thereafter uw1-320-21, and finally uw1-320-22 in this order as printing out "hop", "step", and "jump" at a respective site.
